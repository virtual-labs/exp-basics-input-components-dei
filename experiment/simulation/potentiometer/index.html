Instructions

Objective:

Learn how a potentiometer controls voltage and adjusts LED brightness through analog input.

Steps to Perform:

  1. Click "Start Simulation" to power up the Arduino circuit
  2. Click on the potentiometer knob to activate it
  3. Use the slider to simulate rotating the potentiometer (0-1023)
  4. Observe how the LED brightness changes based on potentiometer position
  5. Higher values → Brighter LED glow
  6. Lower values → Dimmer LED glow

Key Concepts:

  • Potentiometer: Variable resistor - adjusts voltage by rotating the knob
  • Analog Read: Arduino reads voltage (0-1023) from the potentiometer wiper
  • PWM: Pulse Width Modulation controls LED brightness (0-255)
  • Mapping: Converts sensor range (0-1023) to LED range (0-255)
